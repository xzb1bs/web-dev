HTML Introduction

HTML (HyperText Markup Language) is the standard language for creating webpages. It uses elements to structure content and provide functionality.

HTML Elements

HTML elements are building blocks of a webpage. For example:

  • <h1> - Heading
  • <p> - Paragraph
  • <div> - Container

HTML Attributes

Attributes provide additional information about elements. For example:

<a href="https://example.com" target="_blank">Visit Example</a>

HTML Styles

HTML allows inline styles using the style attribute or external stylesheets. Example of inline styling:

This is a styled paragraph.

HTML Lists

HTML supports ordered and unordered lists. Example:

Unordered List:

  • Item 1
  • Item 2
  • Item 3

Ordered List:

  1. Step 1
  2. Step 2
  3. Step 3
